# Tanxz Server
This is the server code for the Tanxz multiplayer game

## Refresh Go Imports

`go mod download`

## Deploy
Locally, run `./deploy.sh`

## Build for Ubuntu
```
env GOOS=linux GOARCH=amd64 go build main.go
```

## Install Go For Mac using Brew
```
brew install go
```
or
```
brew upgrade go
```
